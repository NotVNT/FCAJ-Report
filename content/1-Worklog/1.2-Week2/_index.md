## Week 2 - IAM Roles for EC2 & Introduction to Amazon EC2

### Weekly Topic

IAM review + IAM Roles for EC2 + foundational Amazon EC2 operations

### Weekly Objectives

* Review and consolidate IAM & VPC knowledge from Week 1.
* Deep dive into IAM Roles for EC2 and basic Amazon EC2 operations.

### Tasks to be carried out this week:
| Date | Day | Task | Lab / Project |
| --- | --- | --- | --- |
| 27/04/2026 | Mon | Review IAM: users, groups, roles, and policies.<br>Review VPC components: subnet, route table, Internet Gateway, and security groups.<br>Set up $5 Billing Alarm and review AWS credit usage strategy.<br>Practice all Week 1 labs. | [Lab 000001 - Create your first AWS account](https://000001.awsstudygroup.com/)<br>[Lab 000002 - Manage access with AWS IAM](https://000002.awsstudygroup.com/)<br>[Lab 000003 - Basic networking with Amazon VPC](https://000003.awsstudygroup.com/) |

| 29/04/2026 | Wed | Create VPC and security groups for both Linux and Windows EC2.<br>Launch Amazon Linux 2023 and Windows Server 2025.<br>Connect via SSH and RDP; review Key Pair, AMI, and instance type concepts.<br>Complete Lab 000004. | [Lab 000004 - Introduction to Amazon EC2](https://000004.awsstudygroup.com/) |


### Expected Outcomes

* Consolidate IAM and VPC understanding through hands-on review.
* Set up basic cost controls such as Billing Alarm and credit tracking.
* Understand how IAM Roles for EC2 replace static credentials when accessing AWS services.
* Launch and access both Linux and Windows EC2 instances with proper networking and security configuration.
* Grasp the relationship between AMI, instance type, key pair, and security groups in the EC2 deployment workflow.
